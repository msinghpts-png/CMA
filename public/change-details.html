Description

Deploy critical Microsoft security patches KB5034441 and KB5034439 to all production web servers in the DMZ. This includes 12 Windows Server 2022 instances running IIS.

Business Justification

Address critical vulnerabilities identified in recent security scan. CVE-2024-21351 has a CVSS score of 9.8 and is being actively exploited in the wild.

Implementation Steps

  1. Take VM snapshots of all target servers
  2. Disable servers in load balancer (rolling)
  3. Install patches via WSUS
  4. Reboot servers
  5. Verify services startup
  6. Re-enable in load balancer
  7. Proceed to next batch

Backout Plan

  1. Revert to VM snapshot taken before patching
  2. Uninstall patches if snapshot unavailable
  3. Restore from backup as last resort
