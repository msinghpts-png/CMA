Deploy Q1 Security Patches to Production Web Servers

CHG-000221 · Approved

Overview

Normal Risk: Medium Impact: Medium

Description

Deploy critical Microsoft security patches to all production web servers in the DMZ. Includes 12 Windows Server 2022 instances running IIS.

Business Justification

Addresses critical vulnerabilities identified in recent security scan. CVE-2024-21351 has a CVSS score of 9.8.

Implementation Steps

  1. Take VM snapshots of all target servers
  2. Disable servers in load balancer (rolling)
  3. Install patches via WSUS
  4. Reboot servers
  5. Verify services startup
  6. Re-enable in load balancer

Backout Plan

  1. Revert to VM snapshot taken before patching
  2. Uninstall patches if snapshot unavailable
  3. Restore from backup as last resort
